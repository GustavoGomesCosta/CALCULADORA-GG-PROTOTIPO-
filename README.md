# CLI Core Calculator (JavaScript)

A lightweight, terminal/browser-based calculator built using pure JavaScript. This project focuses on raw logic, algorithms, and modular programming structures without relying on HTML/CSS DOM manipulation.

## 🚀 Features

- **Core Arithmetic:** Addition, Subtraction, Multiplication, and Division (with zero-division safety).
- **Advanced Math:** Power/Exponentiation (`**` operator) and Modulo operations (Even/Odd checker).
- **Statistics:** Dynamic average calculator using array-like logic handling up to 100 custom entries.
- **Percentages:** Two distinct modules to calculate direct values and find percentage ratios.
- **Loops & Matrices:** Dynamic multiplication and division tables printed directly to the developer console.

## 🛠️ Concepts Demonstrated

- JavaScript Functions & Modular Architecture
- Control Flow (`Switch/Case` & `While` loops)
- Data structures and iterative rendering (`For` loops)
- Input type casting and data sanitization
